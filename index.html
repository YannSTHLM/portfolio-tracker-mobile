Formula:

Buy Amount = (Profit ÷ Target Return) - Cost Basis

Where:

  • Profit = Market Value - Cost Basis (can be positive or negative)
  • Target Return = decimal (e.g., 4% = 0.04, -5% = -0.05)
  • Buy Amount = amount to invest today to reach the target return

Use Cases:

  • Positive profit + positive target: "Dilute my return to X%"
  • Negative profit + negative target: "Average down to reach -X% return"
  • Negative profit + positive target: Shows N/A (impossible to reach positive return when underwater)

📊 What It Measures

Each cell compares a shorter-term return against a longer-term return for the same asset. The difference (delta) tells you whether momentum is accelerating or decelerating.

Delta = Short-term return − Long-term return

⏱️ Signal Pairs (Short vs Long)

  • 1W vs 1M Micro momentum · Tactical (days → weeks)
  • 1M vs 3M Near-term trend · Tactical (weeks → quarter)
  • 3M vs 1Y Medium-term momentum · Strategic
  • 1Y vs 3Y Trend establishment · Structural
  • 3Y vs 5Y Long-term regime · Secular

🎨 Color Coding

📈
Green (+0.5pp)
Short-term return exceeds long-term → momentum is accelerating
📉
Red (−0.5pp)
Short-term return lags long-term → momentum is decelerating
➡️
Gray (±0.5pp)
Momentum is neutral — no significant shift

🔢 Composite Score

The average delta across all five signal pairs. A single number summarizing the overall momentum direction.

📈 > +0.5pp — Broad acceleration across timeframes
➡️ ±0.5pp — Mixed or neutral signals
📉 < −0.5pp — Broad deceleration across timeframes

💡 Quick Interpretation Guide

• All green across pairs → strong confirmed momentum, consider holding/adding
• Green short-term + red long-term → recent improvement, trend may be reversing up
• Red short-term + green long-term → still positive long-term but losing steam
• All red across pairs → broad deceleration, consider reducing exposure

📖 How to Read the Momentum-Acceleration Recommendations (click to expand)

Core Idea

This algorithm does not ask "Has this asset gone up?" — it asks "Is this asset's momentum getting stronger or weaker?" across multiple timeframes. An asset is attractive when its shorter-term performance is improving relative to its longer-term performance.

The Five Signals

SignalMeaningType
1W vs 1MIs the last week stronger than the last month?Micro momentum
1M vs 3MIs the last month stronger than the last quarter?Tactical trend
3M vs 1YIs the last quarter stronger than the last year?Medium-term trend (most important)
1Y vs 3YIs the last year stronger than the last 3 years?Structural trend
3Y vs 5YIs the last 3 years stronger than the last 5 years?Long-term regime

Each value is in percentage points (pp): the shorter-period return minus the longer-period return. Positive → momentum accelerating. Negative → momentum decelerating.

Key Columns in the Table

  • Wtd Score — Weighted Momentum Score (0.15×1W + 0.25×1Mvs3M + 0.30×3Mvs1Y + 0.20×1Yvs3Y + 0.10×3Yvs5Y). The 3Mvs1Y carries the most weight because it best captures rebalancing-relevant trends.
  • Br — Breadth Score (number of green signals minus number of red signals). Measures how many timeframes agree on direction. +5 = all accelerating, −5 = all decelerating, 0 = mixed.
  • 3Mvs1Y / 1Mvs3M — The two most actionable signals. Both should be non-red for any buy decision.

Decision Logic (Simplified)

Wtd ScoreBreadthKey SignalDecision
> +1.5≥ +33Mvs1Y greenStrong Buy
> +0.5≥ +21Mvs3M & 3Mvs1Y not redBuy / Increase
> +0.50 to +1Mixed confirmationMild Buy / Watch
±0.5AnyMixed or neutralHold
< −0.5≤ −2Broad weaknessTrim
< −2.0≤ −3Severe weaknessStrong Trim / Sell

Portfolio Role & Exposure Adjustments

  • Cash / Short-Duration (Bucket 1): More forgiving — trim signals are overridden unless momentum is severely negative. Defensive assets are not expected to show strong momentum.
  • Volatile / Thematic (Bucket 3): Stricter — trim signals for energy, metals, and commodities are kept intact because downside risk is higher.
  • Duplicate Exposure: When multiple holdings overlap (e.g., multiple global equity funds), the algorithm flags duplicates. Prefer adding to the strongest/cleanest holding; prefer trimming the weakest/most redundant.

Market Regime Filter

The regime banner at the top shows what fraction of assets are accelerating. When most assets are decelerating (Defensive regime), buy signals are automatically downgraded — the algorithm becomes more conservative. Only the strongest confirmed assets should be bought in a weak market.

Golden Rules

"Do not buy because of one green number. Do not sell because of one red number. Act only when several timeframes agree."
